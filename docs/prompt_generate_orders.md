# 주문 데이터 생성 프롬프트

## 목적
3개 플랫폼(Cafe24, Qoo10, Shopee)의 역직구 주문 데이터를 생성한다.
국가별·채널별·제품유형별 트렌드 분석이 가능한 충분한 볼륨과 현실적 분포가 핵심이다.

## 제약 조건

### 플랫폼-국가 제약 (CHECK 제약조건과 일치 필수)
| 플랫폼 | 허용 국가 |
|--------|----------|
| Cafe24 | KR, JP, SG |
| Qoo10  | JP, SG |
| Shopee | SG |

### 제품 유형 (CHECK 제약조건)
sunscreen, toner, serum, cream, lip

### 브랜드
이니스프리, 토리든, 라운드랩, 스킨푸드, 코스알엑스, 닥터지, 아누아, 메디힐, 비플레인, 달바

### 기간
2025-10-01 ~ 2026-03-25 (약 6개월)

### 볼륨 목표
- **Cafe24**: 400건 (order_id: C24-0001 ~ C24-0400)
- **Qoo10**: 350건 (order_id: Q10-0001 ~ Q10-0350)
- **Shopee**: 250건 (order_id: SHP-0001 ~ SHP-0250)
- **합계**: 1,000건

### 가격 범위 (unit_price_usd)
| 유형 | 범위 |
|------|------|
| sunscreen | 12.00 ~ 28.00 |
| toner | 12.00 ~ 22.00 |
| serum | 18.00 ~ 35.00 |
| cream | 18.00 ~ 32.00 |
| lip | 7.00 ~ 15.00 |

### 수량 (quantity)
1~5, 대부분 1~2

## 분포 규칙

### 국가별 비중
- **JP**: 전체의 ~50% (Cafe24 50% + Qoo10 70%)
- **SG**: 전체의 ~35% (Cafe24 20% + Qoo10 30% + Shopee 100%)
- **KR**: 전체의 ~15% (Cafe24 30%)

### 제품 유형별 비중
- sunscreen: 30% (가장 높음 — 역직구 뷰티의 핵심)
- toner: 22%
- serum: 22%
- cream: 16%
- lip: 10%

### 시계열 트렌드 (현실적 패턴)
- **10~11월**: 기본 물량, cream/serum 비중 높음 (겨울 준비)
- **12~1월**: 선크림 감소, 보습 제품 증가
- **2~3월**: 선크림 급증 (봄/여름 UV 준비), 일본향 선크림 특히 증가
- **연말 (12월)**: 립/선물세트 소폭 증가

### 상품명 규칙
- 실제 이커머스 상품명 스타일
- `[브랜드] [제품라인] [핵심성분/기능] [용량]` 형태
- 간헐적으로 마케팅 문구 포함: "[1+1]", "리뉴얼", "대용량", "기획세트"
- Gold Example의 raw_input과 겹치는 상품명 다수 포함 (추출 시 few-shot 매칭 테스트용)

## 출력 형식
CSV 3개 파일:
- `orders_cafe24.csv`
- `orders_qoo10.csv`
- `orders_shopee.csv`

컬럼: order_id,order_date,destination_country,brand,product_name,product_type,quantity,unit_price_usd
