# MarketPulse 시스템 검증 정답지

> **목적**: 시스템이 완전히 구축된 후, 사용자 질문에 대해 시스템이 올바른 인사이트를 도출하는지 검증하기 위한 정답지.
> 데이터에 의도적으로 심어진 패턴(A~F)을 기반으로, 질문→기대 답변→검증 기준을 정의한다.
>
> **사용법**: 각 스토리를 순서대로 테스트. `검증 기준`이 모두 충족되면 PASS.
>
> **확장 규칙**: 새 스토리 추가 시 아래 템플릿을 복사하여 해당 카테고리에 추가.

---

## 스토리 템플릿

새 스토리를 추가할 때 아래 형식을 따른다:

```markdown
### S-XX. [스토리 제목]

- **카테고리**: 트렌드 감지 | 시장 세분화 | 신규 속성 감지 | 국가 비교 | 인과 추론 | 복합 질의
- **관련 패턴**: A, B, C, D, E, F 또는 신규
- **데이터 근거**: 어떤 데이터에서 이 답이 도출되는지 (테이블, 필터 조건)
- **난이도**: 1 (단순 조회) ~ 5 (복합 추론)
- **선행 스토리**: 없음 | S-XX (이 스토리가 성립하려면 먼저 통과해야 하는 스토리)

**질문**: 사용자가 물어볼 법한 자연어 질문

**기대 답변**: 시스템이 도출해야 하는 핵심 내용 (요약)

**기대 답변 상세**:
> 시스템이 생성해야 하는 답변의 구체적 내용.
> 수치, 비율, 인과관계 등을 포함.

**검증 기준**:
- [ ] 체크리스트 항목 1
- [ ] 체크리스트 항목 2

**호출 예상 도구**:
- MCP 서버 / 쿼리 종류 (어떤 데이터소스를 조회해야 하는지)
```

## 카테고리 1: 트렌드 감지

### S-01. 일본 선크림 시장에서 비건 트렌드 상승 감지

- **카테고리**: 트렌드 감지
- **관련 패턴**: A
- **데이터 근거**: JP + sunscreen 주문에서 "비건" 키워드 포함 비율의 월별 추이
- **난이도**: 2
- **선행 스토리**: 없음

**질문**: 최근 6개월간 일본 선크림 시장에서 주목할 만한 트렌드가 있나요?

**기대 답변**: 일본 선크림 시장에서 "비건" 속성 제품의 비율이 지속적으로 상승 중

**기대 답변 상세**:
> 2025년 10월 약 19%였던 비건 선크림 비율이 2026년 3월 56%로 약 3배 상승했습니다.
> 매월 단조 증가 추세를 보이고 있어, 일본 소비자들의 비건 선케어 선호가 뚜렷하게 강화되고 있습니다.
> 특히 2026년 1분기(1~3월) 들어 50%를 돌파하며 주류 트렌드로 자리잡는 모습입니다.

**검증 기준**:
- [ ] "비건" 키워드의 상승 트렌드를 언급
- [ ] 대략적인 수치 범위 (초기 ~20%, 최근 ~55% 이상) 제시
- [ ] 단조 증가(monotonic increase)임을 인식
- [ ] 일본 시장 한정임을 명시

**호출 예상 도구**:
- 주문 데이터 MCP: JP + sunscreen 필터 → 월별 "비건" 포함 비율 집계

- [ ] 하락이 일관적(monotonic)임을 인식
- [ ] (보너스) 비건 상승과의 상관관계 언급

**호출 예상 도구**:

**기대 답변 상세**:
> 싱가포르 선크림 주문의 71~80%가 "워터프루프" 속성을 포함하고 있으며, 이 비율은 6개월간 변동 없이 안정적입니다.
> 이는 싱가포르의 열대 기후(고온다습, 잦은 소나기)가 워터프루프를 기능적 필수 요건으로 만들기 때문입니다.
> 싱가포르 진출 시 워터프루프 미포함 선크림은 경쟁력이 매우 낮을 것으로 예상됩니다.

**검증 기준**:
- [ ] "워터프루프"가 지배적 속성임을 언급
- [ ] 비율 70~80% 범위 제시
- [ ] 시계열 변화 없이 일관됨을 인식 (트렌드가 아닌 구조적 선호)
- [ ] (보너스) 기후 요인과의 인과관계 언급

**호출 예상 도구**:
- 주문 데이터 MCP: SG + sunscreen 필터 → 월별 "워터프루프" 포함 비율
- Neo4j KG: SG → ClimateZone → SkinConcern 인과 체인

## 카테고리 2: 시장 세분화 / 블루오션

### S-04. 일본 선크림 시장의 블루오션 기회 발견

- **카테고리**: 시장 세분화
- **관련 패턴**: D
- **데이터 근거**: JP + sunscreen에서 속성 조합별 주문량 / 상품 다양성 / 반복 구매율
- **난이도**: 4
- **선행 스토리**: S-01, S-02

**질문**: 일본 선크림 시장에서 경쟁이 적으면서 수요가 있는 틈새 시장이 있나요?

**기대 답변**: "비건 + 무기자차" 조합이 블루오션 — 1개 상품에 수요가 집중, 높은 반복 구매율

**기대 답변 상세**:
> 일본 선크림 시장을 속성 조합별로 분석하면 세 가지 뚜렷한 세그먼트가 보입니다:
>
> 1. **레드오션 — 톤업 단독**: 48건, 18개 상품. 모든 브랜드가 참여한 과밀 시장.
> 2. **블루오션 — 비건+무기자차**: 11건이지만 단 1개 상품(이니스프리)에 집중.
>    평균 주문 수량 3.0개로 높은 반복/대량 구매 패턴. 공급은 적은데 수요는 충성도 높음.
> 3. **기회 영역 — 비건+톤업**: 28건, 3개 상품. 두 트렌드의 교집합으로 성장 잠재력 있음.
>
> **추천**: "비건+무기자차" 조합의 신규 상품 출시가 유리.
> 현재 이니스프리 1개 상품이 독점하고 있어 진입 여지가 큽니다.

**검증 기준**:
- [ ] 세 세그먼트(레드오션, 블루오션, 기회)를 구분
- [ ] 블루오션의 핵심 특성: 적은 상품 수(1~2개) + 높은 반복 구매
- [ ] 레드오션의 핵심 특성: 많은 상품 + 많은 브랜드 참여
- [ ] 블루오션 진입 기회를 제안
- [ ] (보너스) 기회 영역(비건+톤업)의 성장 잠재력 언급

**호출 예상 도구**:
- 주문 데이터 MCP: JP + sunscreen → 속성 조합별 그룹화 → 주문 건수, 고유 상품명 수, 평균 수량

### S-05. "왜 비건+무기자차 조합이 블루오션인가?" — 인과 추론

- **카테고리**: 인과 추론
- **관련 패턴**: D + Neo4j 온톨로지
- **데이터 근거**: Neo4j 인과 체인 (JP → Climate → SkinConcern → Function → Ingredient) + 주문 데이터
- **난이도**: 5
- **선행 스토리**: S-04

**질문**: 왜 일본에서 비건 무기자차 선크림에 대한 수요가 있는 건가요?

**기대 답변**: 일본의 UV 환경 + 비건 소비 트렌드가 결합하여 "무기자차(징크옥사이드) + 비건" 수요를 만듦

**기대 답변 상세**:
> 인과 체인:
> 1. **환경**: 일본(JP)은 "온대 고UV 여름" 기후대 → UV 손상 우려가 높음
> 2. **기능 수요**: UV 손상 → UV 차단 기능 수요 (strength 0.92)
> 3. **성분**: UV 차단 ← 징크옥사이드(무기자차의 핵심 성분) (HAS_FUNCTION)
> 4. **가치 트렌드**: 비건 소비 트렌드가 18% → 58%로 급성장 중
> 5. **결합**: 환경적 필요(무기자차) + 가치 선호(비건)가 결합 → 비건 무기자차 선크림 수요
>
> 하지만 현재 이 조합을 제공하는 상품은 이니스프리 1개뿐이어서 블루오션이 형성됨.

**검증 기준**:
- [ ] Neo4j 인과 체인을 통한 환경→기능→성분 연결 제시
- [ ] 비건 트렌드 상승 데이터 인용
- [ ] 두 요인의 결합이 수요를 만든다는 논리 전개
- [ ] 공급 부족(1개 상품)과 수요 존재를 대비

**호출 예상 도구**:
- Neo4j KG: JP → HAS_CLIMATE → TRIGGERS → DRIVES_DEMAND → HAS_FUNCTION 체인
- 주문 데이터 MCP: "비건" 트렌드 + "무기자차" 상품 수 조회

## 카테고리 3: 신규 속성 감지

### S-06. 마이크로바이옴 신규 속성 출현 감지

- **카테고리**: 신규 속성 감지
- **관련 패턴**: E
- **데이터 근거**: 전체 toner/serum 주문에서 "마이크로바이옴" 키워드 월별 건수
- **난이도**: 3
- **선행 스토리**: 없음

**질문**: 최근에 새롭게 등장하거나 급성장하는 성분/속성이 있나요?

**기대 답변**: "마이크로바이옴"이 2026년 1월부터 출현, 3월에 25건으로 급증 — 승격 후보

**기대 답변 상세**:
> "마이크로바이옴" 키워드가 2026년 1월에 처음 3건 등장한 후, 2월 8건, 3월 25건으로 급성장했습니다.
> 2025년 10~12월에는 0건이었으므로, 완전히 새로운 트렌드입니다.
>
> 특성:
> - 브랜드: 토리든(Torriden) 단독
> - 제품 유형: 토너/세럼
> - 3월 기준 25건으로 자동 승격 임계치(20건) 돌파
>
> **권장 조치**: 이 속성을 정식 속성으로 승격하고, 다른 브랜드의 유사 제품 출현 여부를 모니터링해야 합니다.

**검증 기준**:
- [ ] "마이크로바이옴"을 신규 출현 속성으로 식별
- [ ] 2026-01 시작, 월별 급증 패턴 (0→0→0→3→8→25) 인식
- [ ] Torriden 단독 브랜드임을 언급
- [ ] 임계치(20건) 돌파를 근거로 승격 권장
- [ ] (보너스) 향후 모니터링 필요성 언급

**호출 예상 도구**:
- 주문 데이터 MCP: toner+serum 전체 → "마이크로바이옴" 키워드 월별 집계
- (자동화) EmergingAttribute 테이블 조회 또는 additionalAttrs 빈도 분석

## 카테고리 4: 국가별 비교 분석

### S-08. 국가별 토너/세럼 성분 선호 차이

- **카테고리**: 국가 비교
- **관련 패턴**: F
- **데이터 근거**: 국가별 toner+serum에서 성분 키워드 포함 비율
